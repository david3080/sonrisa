リーサという言葉は、スペイン語で「笑い」「笑顔」を表します。株式会社リーサは、温故知新の精神で最先端のテクノロジーとそれらを実生活にいかすノウハウをもとに人々を笑顔にするソリューションを提供します。

提供するサービス


1. フィンテック関連のコンサルティング
2. API 戦略とAPI プラットフォームの構築
3. スマホアプリデザイン＆アプリ開発
4. OIDCやFIDO、WebAuthnを使った認証認可システム構築
5. 新規ビジネスの立ち上げのコンサルティング
6. テクノロジーの民主化によるスマートシティの実現
7. 実践的なテクノロジー教育
8. 実践的な人間中心設計やDX
9. 論語や歎異抄といった東洋哲学の基礎と実験にもとづく進化認知科学
10. 太陽光発電事業をはじめとするSDGs事業

FINTECH

フィンテックのビジネスモデルや守るべきルール、また守るべきセキュリティとそれを担保する業界標準に関するコンサルと開発サービスを提供します。

API

APIとはシステムのモジュール性を高め、サービスがより多くのユーザやシステムから利用されるようにするしくみ。このAPIに共通して必要な機能のコンサルとプラットフォームを提供します。

Smartphone

ネイティブアプリは操作性のよさと引換えにiOS/Android用に別々にコストをかけて開発しているのが現状です。弊社ではFlutterを使い安価に効率的にマテリアルデザインベースのリッチなネイティブアプリ開発を体系化しており、それを使った企画や開発組織の立ち上げを支援します。

OIDC/FIDO

昨今マイクロサービスやAPIを組み合わるシステム構築が当たり前になってきています。そのような分散型システムではトークンベースの認証認可、その業界標準であるOIDCの利用が適しています。またパスワードレスを実現する生体認証標準にFIDO2があります。弊社ではこれらのサービスを提供します。
