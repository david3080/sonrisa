株式会社リーサは、温故知新の精神で最先端のテクノロジーとそれらを実生活にいかすノウハウをもとに人々を笑顔にするソリューションを提供します。

FINTECH

フィンテックのビジネスモデルや守るべきルール、また守るべきセキュリティとそれを担保する業界標準に関するコンサルと開発サービスを提供します。

API

APIとはシステムのモジュール性を高め、サービスがより多くのユーザやシステムから利用されるようにするしくみ。このAPIに共通して必要な機能のコンサルとプラットフォームを提供します。

Smartphone

ネイティブアプリは操作性のよさと引換えにiOS/Android用に別々にコストをかけて開発しているのが現状です。弊社ではFlutterを使い安価に効率的にマテリアルデザインベースのリッチなネイティブアプリ開発を体系化しており、それを使った企画や開発組織の立ち上げを支援します。

OIDC/FIDO

昨今マイクロサービスやAPIを組み合わるシステム構築が当たり前になってきています。そのような分散型システムではトークンベースの認証認可、その業界標準であるOIDCの利用が適しています。またパスワードレスを実現する生体認証標準にFIDO2があります。弊社ではこれらのサービスを提供します。

学び

渋沢栄一「論語と算盤」

2021年コロナ禍で三密をさけるだのリモートワークだのデジタル化だのと騒がれていますが、いまから約180年前にお生まれになった渋沢栄一先生の著書「論語と算盤」を題材に新たな時代を生き抜くための普遍的な考え方を学びます。

ITスキルが十分ではない学生が対面サポートなしのオンデマンド型授業でスマホアプリのプロトタイピングを行うことができるアクティブラーニング型演習授業を研究開発し、授業実施した話を愛知産業大学の造形学研究所報に論文投稿しました。

講演

テクノロジーの民主化

最近はサイバーとフィジカルを融合させ人間中心社会を目指すSosiety5.0の実現に有効とされるIoTやAIといったテクノロジーを安価で気軽にはじめられるようになっています。また市民自身がテクノロジーを活用して行政や社会の課題を解決する「シビックテック」といった取り組みも行われています。これら「テクノロジーの民主化」というテーマについて、21世紀交流サロン・葵丘にて「AI技術最前線からの報告 Part2」として講演を行いました。
